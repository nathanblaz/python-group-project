## [Features](https://github.com/nathanblaz/showboat-group-project/wiki/Feature-List)

**User Log In / Log Out**

* Users can sign up, log in and log out.

**Photos**

* Logged in users can post photos.
* Logged in and non-logged in users can view photos.
* Logged in users can edit their own photo details.
* Logged in users can delete their own photos.

**Albums**

* Logged in users can create photo albums.
* Logged in and non-logged in users can view photo albums.
* Logged in users can edit their own albums.
* Logged in users can delete their own albums.

**Comments**

* Logged in users can post comments on photos.
* Logged in and non-logged in users can view comments.
* Logged in users can edit their own comments.
* Logged in users can delete their own comments.

**Tags**

* Logged in users can apply tag photos.
* Logged in and non-logged in users can view photos of a specific tag.
* Logged in users can edit their own tags.
* Logged in users can delete their own tags.
